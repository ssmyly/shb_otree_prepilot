Step 3: Hiring and Wages
1
Your signal = task score + training bonus + a small random noise (between {{ noise_min }} and +{{ noise_max }} points). The noise represents randomness in how employers perceive your performance.
2
Hiring threshold = {{ threshold }} points. If your signal ≥ {{ threshold }}, you are HIRED and receive a wage. If signal < {{ threshold }}, your wage this round = 0.
3
If hired, your wage is computed by a formula based on your signal(s):
{% if is_shb %} Your wage formula (SHB condition):
wage = {{ prior_mean }} + {{ kappa_1 }} × (your signal this round − {{ prior_mean }})

Example: If your signal = {{ example_signal }}:
wage = {{ prior_mean }} + {{ kappa_1 }} × ({{ example_signal }} − {{ prior_mean }}) = {{ example_w_shb }} ECU {% else %} Your wage formula (No-SHB condition):
wage = {{ prior_mean }} + κ_n × (average of all your signals − {{ prior_mean }})

where κ increases as you complete more rounds:
  Round 1: κ₁ = {{ kappa_1 }}
  Round 2: κ₂ = {{ kappa_2 }} (employer uses 2 signals → more precise)
  Round 3: κ₃ = {{ kappa_3 }} (employer uses 3 signals → most precise)

Example (Round 2): If your Round 1 signal = 60 and Round 2 signal = {{ example_signal }}:
Average signal = (60 + {{ example_signal }}) / 2 = 62.5
wage = {{ prior_mean }} + {{ kappa_2 }} × (62.5 − {{ prior_mean }}) = {{ example_w_noshb_r2 }} ECU {% endif %}

Step 4: Earnings
  • Round net earnings = wage − training cost (floored at 0)
  • Total bonus = sum of all round net earnings
  • Total payment = ${{ participation_fee_usd }}.00 base + (total ECU ÷ {{ ecu_per_usd }})

Example: If you earn 48 ECU net across 3 rounds: bonus = 48 ÷ {{ ecu_per_usd }} = $4.80. Plus ${{ participation_fee_usd }}.00 base = $7.80 total.

Summary of the experiment flow:
  1. Investment decision (choose 0–3 training tokens)
  2. Complete counting task (2 minutes)
  3. See your signal, hiring outcome, and wage
  4. Repeat for {{ num_rounds }} rounds total
  5. Answer short exit survey
